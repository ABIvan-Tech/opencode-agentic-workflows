# Self-Improvement Harness

This folder holds the policy, schemas, synthetic scenario manifests, and release metadata for the bounded self-improvement loop.

## Scope

The harness is intentionally limited to the allowlisted low-risk overlay namespace and must not be used to alter protected control-plane prompts, permissions, routing, or governance.

## Contents

- `policy/` machine-readable allowlist and promotion policy
- `schemas/` JSON schemas for candidates, evaluations, and releases
- `scenarios/` synthetic scenario definitions and manifest
- `reports/` evaluation report templates
- `releases/` accepted-release metadata
